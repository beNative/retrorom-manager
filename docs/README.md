# RetroRom Manager

RetroRom Manager is a cross-platform tool designed to help you manage your EmulationStation ROM collections. It allows you to scan for issues, clean up media, and keep your gamelists synchronized.

## Features

- **Scan Systems**: Automatically detect ROMs and media in your EmulationStation directories.
- **Fix Issues**:
    - **Sync Gamelist**: Add missing ROMs to `gamelist.xml`.
    - **Link Media**: Automatically link existing images and videos to ROMs.
    - **Clean Media**: Remove orphaned media files that don't have corresponding ROMs.
- **Frameless UI**: A modern, sleek interface inspired by VSCode.
- **Keyboard Navigation**: Fully accessible via keyboard.

## Installation

Download the latest release for your operating system from the [Releases](https://github.com/yourusername/retrorom-manager/releases) page.

## Usage

1. Launch the application.
2. Select your root ROMs folder (e.g., `/home/pi/RetroPie/roms`).
3. Select a system from the sidebar.
4. Use the dashboard to view statistics and fix issues.
